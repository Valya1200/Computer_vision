## 1.2 Фильтр Гаусса

Основная задача гауссовской фильтрации — сглаживать изображение, уменьшая шум и устраняя ненужные детали, которые могут мешать анализу. С математической точки зрения обработка изображения фильтром Гаусса есть свертка изображения с гауссовским ядром

**Влияние параметра стандартного отклонения на характеристики гауссовского ядра**

В представленной таблице приведены результаты применения гауссовского фильтра с ядром размером 5×5 при различных значениях стандартного отклонения σ.
При σ = 1.0 на 3D-визуализации наблюдается выраженный центральный пик, что свидетельствует о высокой концентрации весов в окрестности центрального элемента ядра. Такая конфигурация обеспечивает минимальное размытие изображения с сохранением значительной части высокочастотных деталей.

При σ = 10.0 центральный пик отсутствует, а веса элементов ядра распределены практически равномерно. Тепловая карта подтверждает данную закономерность: градиент цветовых оттенков минимален, что соответствует значительному сглаживанию и выраженному размытию изображения за счёт равновесного вклада всех соседних пикселей.

## 3.3 поворот изображений на угол кратный 90 градусов
Операция поворота предназначена для изменения ориентации изображения на заданный угол (90°, 180°, 270°) относительно исходного положения. Поворот выполняется без интерполяции с сохранением точных значений пикселей.

Для выполнения операций поворота изображения на углы 90°, 180° и 270° применялась встроенная функция библиотеки NumPy np.rot90().
Описание функции np.rot90(img, k, axes=(0,1))
