Таблица 1 – Результаты использования медианного фильтра


Для анализа характера шума построены профили интенсивности пикселей 56-й строки изображения в пространстве столбцов. Разложение изображения на RGB-каналы выполнено с помощью cv2.split(), визуализация профилей реализована посредством библиотеки matplotlib.pyplot.
Профиль канала G оригинального изображения демонстрирует характерные для импульсного шума "соль-перец" резкие переходы интенсивности от 0 до 255, соответствующие случайным выбросам белых ("соль") и чёрных ("перец") пикселей.
Применение медианного фильтра с ядром 3×3 значительно снижает амплитуду импульсов за счёт замены выбросных значений медианными значениями локального соседства, однако единичные всплески сохраняются.
Увеличение размера ядра до 5×5 обеспечивает полное устранение импульсного шума: все выбросы заменяются репрезентативными значениями расширенного соседства (25 пикселей), что подтверждается монотонным гладким профилем интенсивности.
Фильтр Гаусса
Основная задача гауссовской фильтрации — сглаживать изображение, уменьшая шум и устраняя ненужные детали, которые могут мешать анализу. С математической точки зрения обработка изображения фильтром Гаусса есть свертка изображения с гауссовским ядром
