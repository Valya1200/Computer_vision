# Лабораторная раюота №2 Визуальная одометрия (навигация)

Цель: Разработать систему визуальной одометрии (навигации) по группе фотографий.

# ШАГ 1: Детектирование ключевых точек и вычисление SIFT дескрипторов
## 1.1. Предобработка данных
Загружено 6 последовательных изображений размером 320×240 пикселей. Конвертация в градации серого для ускорения обработки

## 1.2. Детектирование ключевых точек (Harris Corner Detector)
Алгоритм:
1. Вычисление градиентов Ix, Iy с помощью свертки Собеля
2. Построение матрицы отклика углов: M = [[Ixx, Ixy], [Ixy, Iyy]]
3. Вычисление минимального собственного значения матрицы M
4. Поиск локальных максимумов отклика
5. Выборка топ-N углов (N=30)
Результат: Найдено 180+ ключевых точек по всем изображениям

# ШАГ 2: Сопоставление точек

## 2.1 Вычисление дескрипторов (SIFT-подобные)
Для каждой ключевой точки (16×16 патч):
1. Вычисление градиентов gx, gy в патче
2. Разбиение на 4×4 блоков (по 4×4 пикселя)
3. Построение 8-биновой гистограммы ориентаций в каждом блоке
4. Нормализация L2 + клиппинг (0.2) + перенормировка
5. Итоговый дескриптор: 128-мерный вектор
Результат: 100+ валидных дескрипторов на изображение


#  ШАГ 3: Построение траектории
## 3.1 Сопоставление признаков (Lowe's ratio test)
Критерий: dist1 < 0.75 × dist2 (где dist1, dist2 — ближайшие соседи)
Результат: 20-50 совпадений между соседними кадрами

# ШАГ 4: Визуализация траектории
## 4.1 Оценка движения камеры (RANSAC)
1. Инициализация: 100 итераций RANSAC
2. Модель: аффинное преобразование (только трансляция)
3. Порог inlier: 10 пикселей
4. Аккумуляция смещений → траектория
